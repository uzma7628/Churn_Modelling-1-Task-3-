# Churn_Modelling-1-Task-3-
Predicting customer churn using data analysis to identify which customers are likely to leave the bank."
## Task Objective
The goal is to analyze customer data (Churn Modelling) to understand why customers leave the bank and visualize the key factors affecting churn.

## My Approach
* **Data Inspection:** Dataset load kiya aur columns jaise Age, CreditScore, aur Balance ko check kiya.
* **EDA & Visualization:** Seaborn use karke 'Exited' column (churn) ka distribution analyze kiya.
* **Customer Profiling:** Dekha ke kis age group ya region ke log zyada bank chhor rahe hain.
* **Data Preparation:** Model training ke liye categorical values ko handle kiya.

## Results & Insights
* **Key Factors:** Age aur Balance churn rate par bada asar dalte hain.
* **Visuals:** Heatmaps aur Bar plots ne successfully show kiya ke active members ke chhorne ke chances kam hain.
